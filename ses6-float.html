การจัดวางเลย์เอาท์บนเว็บเพจด้วย Float

    Float กำหนดให้อิเลเมนต์สามารถลอยอยู่ด้านใดด้านหนึ่ง
  • left, right (ให้ลอยอยู่ทางด้านซ้ายและขวาตามลำดับ)
  • inherit ลอยตาม parent element
  • none ไม่ให้มีการลอย (default)
    clear ไม่อนุญาติให้มีการลอยของอิลิเมนต์ค่าที่กำหนดได้คือ
  • left, right (ไม่อนุญาติให้มีการลอยทางด้านซ้ายขวา)
  • both ไม่อนุญาติให้ลอยทั้งสองด้าน
